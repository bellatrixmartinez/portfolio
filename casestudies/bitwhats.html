Bit Whats

User Experience Flows, UI Design, Sketch Prototype, Frontend Implementation, Responsive Design (MaterializeCSS), GIT, Project Management

An application that allows users to send and receive Bitcoin payments through the Lightning Network by sending a text message in Whatsapp.

DISCOVERY. LOW FIDELITY MOCKUPS.

The client approached me with the functioning prototype of the application integrated with WhatsApp and Lightning Network for payments.

His company had created this prototype and needed a landing page explaining the service. They also needed the UI design for the backend solution.

The application serves two main user roles the "Front user", the user that will interact with the whatsapp client and the "Admin" the user who will see all recorded data from the backend.

USER INTERFACE DESIGN.

For the user interface design we went with a clean layout with flat colors.

Our landing page needed to do 2 major things: Have a Call to Action to get users to send a text to get signed up and explain in easy steps how the application works to the users.

We chose Avenir as our font.

For the backend we chose colors that also gave the user the sentiment of the status of the payment so that they can review when needed.

PROTOTYPE AND IMPLEMENTATION.

I designed the UI and built the prototype in Sketch. This was a great tool to collaborate with the client and the development team.

Once all design views were approved I implemented them in the Rails application using Slim. We based our implementation in the Materialize CSS framework so that the landing page would be fully responsive.

The project was a success. The application now processes hundreds of payments daily using Whatsapp and the Lightning Network.
