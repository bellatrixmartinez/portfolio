Yotta

Branding, User Experience Flows, UI Design, Figma Prototype, Webflow, Responsive Design, Project Management

Yotta is the fastest fiber optics internet provider in Sierra Leone.

IDEA AND BRANDING.

We wanted to create a modern corporate yet whimsical branding. The Y represents a playful treatment to the corporate logo also resembling the freedom of a fast internet service.

The blue color represents corporate, freedom and technology.

USER EXPERIENCE.

Met with the client and we went through their ideas and needs for the project.

They were not ready to subscribe people over the internet so they just wanted for people to learn about the plans they offer and contact them.

It was also super important for them to allow users to apply for subsidized internet.

The website is set up so that users are able to search whether the fiber optics service is available in their area.

USER INTERFACE DESIGN.

We defined a modern and bold look and feel. The overall look is dark mode exposing the current situation on Africa's connectivity school system and how we can help solve it.

Big picture, bold fonts. We are using Roboto in different weights.

IMPLEMENTATION.

Designed the prototype in Figma. Implemented the website in Webflow.

Search was implemented using Webflow search.
