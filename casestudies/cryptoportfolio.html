Crypto Portfolio

User Experience Flows, UI Design, Figma Prototype, Frontend Implementation (ELM), Responsive Design, GIT, Project Management

Coin market portfolio management. Add a new coin ticker and track its growth.

IDEA AND USER EXPERIENCE.

Client had an idea of creating a platform where users could add their crypto buys, track their progress and track their gains or losses.

I met with the client a couple of times and understood the visualization part of the application was rather important to inform the user his positions status. The app also needed to be fairly quick in returning information.

For this application we had two types of users. The Customer (logger) and SuperAdmin (client’s company).

Even though this is a web application it was also an important requirement for it to be fully functional and easy to use through any mobile device.

USER INTERFACE DESIGN.

Client brief in terms of design was very open. They did not have a brand or style set up yet.

I went with a dark mode overall look and feel. Clean layouts and flat design across the interface design. Chose Avenir as our font for its easy to read look and simplicity. As far as color palette I went with a few neutral greys, green for positive and red for negative status.

Figma was my software of choice for creating the UI design and components as well as the inicial prototype. This tool allowed me to create all necessary interactions, collaborate with the client and development team and export all necessary components for implementation.

IMPLEMENTATION.

I helped the team with the front end implementation using ELM. By no means I am an ELM expert but it was easy to pick up abs it actually allows you to manage all front end views in an organized and easy way.

The backend was created in Rails. The team also connected the application with coinmarketcap.com API so that the data of the current coin’s price was up to date and accurate.

This application design and development was a complete success. We were able to launch it within the established timeframe and client was very happy with the outcome.

It currently helps thousands of crypto investors on their daily decision making generating these users millions in return.
