1845

User Experience Flows, UI Design, Figma Prototype, Webflow, Responsive Design, Project Management

A website to promote and sponsor children connectivity in Sierra Leone.

IDEA AND BRANDING.

1845 is the year the school that leads the project was founded. It is an important year mark for the community.

We decided to go with a bold and modern color palette. The logo is using Montserrat.

USER EXPERIENCE.

We went through the project's goals and needs. We mapped some informational sections always with the goal for the user to donate. They weren't ready to accept payments online yet but wanted to capture the lead and add then to a sales funnel.

USER INTERFACE DESIGN.

We defined a modern and bold look and feel. The overall look is dark mode exposing the current situation on Africa's connectivity school system and how we can help solve it.

Big picture, bold fonts. We are using Roboto in different weights.

IMPLEMENTATION.

Designed the website in Figma. Implemented the website in Webflow.

One of the requirements of the project is that it needed to be by invitation only. It needed to be password protected. We created an auth landing page where invited users can log in with a password and new users can apply if interested in the project.
